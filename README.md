# HiDoctor

- git clone https://github.com/PwS/HiDoctor

- yarn android

Common Error

```Failed to install the app. Make sure you have the Android development environment set up: https://reactnative.dev/docs/environment-setup.```

- cd android && ./gradlew clean

- chmod +x gradlew
